# scratchProj

**Problem:**
Remove the work of assigning Secret Santas and maintain anonymity.

**Solution:**

# To Run Application:

1.Clone this reposity
2.Install dependencies :
    - Webpack
    - Babel
    - Typescript and its loaders
    - React
    - Express

```npm install```
        
3.Start the application:
```npm start```
4.Navigate to the Backend: - Opens Server

```cd backend```
```npm server```
